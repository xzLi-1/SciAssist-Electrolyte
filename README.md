# 🔋 电解液研究智能助手

基于人工智能的电池电解液研究分析与推荐系统。

## 🎯 项目特色

- **📄 智能文献处理** - 自动提取PDF论文中的电解液配方和性能数据
- **🔍 专业检索系统** - 基于领域知识的智能文献检索
- **🧪 配方分析** - 专业电解液配方分析和性能预测
- **💡 智能建议** - 基于研究的改进建议和风险评估
- **🌐 友好界面** - 直观的Web界面，无需编程经验
